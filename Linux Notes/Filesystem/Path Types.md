
## Absolute Path

When something starts with `/` it is referencing the `root` directory
Examples:
```
/var/ftp/fubar
cd /etc/samba.smb.conf
cd /boot/grub/grub.conf
```


## Relative path

Examples:
```
cd log/debug.txt
```
